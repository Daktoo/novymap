The source code of Novymap.
Forked from Novymap-QVH. Novymap-QVH is a fork of Pigwin's original Novymap.

Based on the original Novymap by Pigwin. Uses LiveAtlas on top of Novylen's Dynmap. Currently hosted by Dakto after QVH handed it over.

Contact us on our Discord if you're interested in becoming a Novymap admin.

The map has two different views, 2D (flat) and 3D (surface).
